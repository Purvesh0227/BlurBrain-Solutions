## ğŸ”§ Troubleshooting

### PHP Not Found
If you get "php is not recognized":
- Use the full path: `C:\xampp\php\php.exe`
- Or add PHP to your system PATH

### Port Already in Use
If port 8000 is busy:
```bash
php -S localhost:8001
```

### Images Not Loading
- Check your internet connection (images are loaded from Unsplash)
- Replace with local images if needed

### Styles Not Applying
- Ensure `styles.css` is in the same directory
- Check browser console for errors
- Clear browser cache
